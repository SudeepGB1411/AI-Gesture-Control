# AI Gesture Controller

An optimized, real-time gesture control system that allows users to control their mouse and interact with their computer using hand gestures via a webcam. 

## Features
- **Mouse Mode Engine:** Optimized cursor movement with median filtering, low-pass smoothing, and edge boosting for precision.
- **Real-time Tracking:** Uses MediaPipe and OpenCV for high-performance hand landmark detection.
- **GUI Overlay:** Built with Tkinter and PIL for visual feedback and heads-up display (HUD).
- **Customizable Sensitivity:** Adjustable gains, deadzones, and smoothing factors.

## Prerequisites
- Python 3.8+
- Webcam
